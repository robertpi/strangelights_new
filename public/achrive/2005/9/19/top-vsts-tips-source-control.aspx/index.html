If you want use the source control get to know the new command line tool ‘h’. It seems what you can do though the IDE is a bit limited, for example there doesn’t seem to be away to map source control folder to a folder on the disk without using ‘h’./o:p

 /o:p

The full details of the commands are available here:/o:p

http://blogs.msdn.com/buckh/articles/CommandLineSummary.aspx/o:p

 /o:p

And if you did want to map a folder the syntax would be:/o:p

h workfold /map /workspace:MyWorkspaceName $/MySSPath C:\MyPath/o:p

 /o:p

There are a couple of slightly annoying things:/o:p

 - There’s no way copy and past the path so you have type it out in manually/o:p

 - Once you’ve mapped the path you can’t just “check out” and start editing, you have to get latest version first. (All right, this is only a small thing but its very annoying if you have to spend time wondering why “check out” is still greyed out)/o:p

 /o:p
